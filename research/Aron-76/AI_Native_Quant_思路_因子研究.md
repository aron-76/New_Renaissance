# AI Native Quant 思路 — 因子研究

## 一、核心思路

用 LLM 读研报理解经济学逻辑，为因子搜索指定方向，再用遗传规划等方法在该方向上搜索因子变种。

## 二、LLM 在量化中的定位

经过逐步讨论：

| 方向 | 初步判断 | 理由 | 备注 |
|------|----------|------|------|
| LLM 直接预测股价 | 不可行 | 未来取决于未来的信息 | |
| LLM 模拟市场来预测 | 待验证 | 模拟得像不等于能预测 | 见市场模拟文档 |
| LLM 生成算子组合因子 | 效率可能低于 GP | 纯组合搜索是 GP 的强项 | 但 LLM 的组合有逻辑，GP 的组合是盲搜，两者性质不同 |
| LLM 处理非结构化数据 | 有价值 | LLM 能理解文本语义，传统 NLP 做不到 | 数据泄露影响待评估 |
| LLM 为搜索指定方向 | 值得尝试 | 方向 × 搜索 | 需要实验验证 |

注：以上判断基于讨论推理，未经实验验证。

## 三、研报作为搜索方向的来源

### 基本思路

- 量化研报描述了某个方向存在 alpha 的逻辑（如"订单簿失衡可以预测短期价格"）
- 研报中给出的具体因子公式可能已经被广泛使用，alpha 衰减（但不能断言"已被挖完"）
- 因子空间在实践中虽有限制（如树深度），但组合数量极大，同一方向上仍可能存在未被发现的有效变种
- 因子空间与物理矿区的类比有局限：矿区面积有限，因子变种数量在实践约束下虽有限但极大

### 研报的性质

研报的发布者（券商研究员、学术研究者）和因子的使用者（基金公司）通常不是同一群体。公开研报中的因子可能面临拥挤问题，但拥挤程度因研报影响力和因子容量而异。

## 四、流程（设计提案）

```
研报 PDF → LLM 读懂逻辑
  → 输出: 相关字段、相关算子、约束条件、时间窗口等
  → 缩小搜索空间

搜索算法（如 GP）在缩小后的空间内搜索
  → 回测验证
  → 筛选有效因子

规模化: 多篇研报 → 多个搜索方向 → 并行搜索 → 因子库
```

LLM 输出的内容可能包括：
- 相关字段（如 bid_volume、ask_volume、spread、vwap 等）
- 相关算子（如 ratio、diff、rank、log、mean_T、std_T 等）
- 约束条件（如"因子应包含买卖量相关字段"）
- 时间窗口（如 1、5、10、30、60 分钟）

注：这是一种简化的流程描述。QuantaAlpha 等已有工作表明，LLM 可以更深度地参与因子构造的迭代过程（生成、回测、分析失败原因、改进），而不仅是提供初始方向。具体采用哪种方式需要实验对比。

## 五、关于数据泄露

LLM 的训练数据包含量化相关文本，存在数据泄露的可能性。但在"LLM 指方向 + GP 搜变种"的场景下，影响可能较小：

- LLM 训练数据中量化相关内容占比很低
- GP 搜出来的变种因子是新的组合，不太可能直接出现在训练数据中
- 最终需要通过严格的样本外回测验证因子有效性

但不能断言"数据泄露不存在"。这个问题需要在实践中持续关注，尤其是在 LLM 直接生成完整因子公式（而非仅指方向）的场景下。

## 六、搜索方法

显式因子（可解释的公式）搜索方法：

| 方法 | 特点 | 备注 |
|------|------|------|
| 遗传规划 (GP) | 启发式搜索，通过变异+交叉+选择演化表达式树 | 最常用，但不保证找到最优解 |
| 符号回归 | GP 是其常见实现方式，也有基于神经网络的方法（如 PySR） | 与 GP 有重叠，不是并列关系 |
| 蒙特卡洛树搜索 (MCTS) | 把因子构造看成序列决策，平衡探索和利用 | 有相关论文（如 AlphaGen） |
| 强化学习 (RL) | Agent 学习构建表达式的策略 | 有相关论文（如 AlphaAgent） |

深度学习（MLP、LSTM、Transformer 等）产出的是隐式因子（黑盒），可解释性低。显式因子和隐式因子各有优劣，不应简单地否定某一方向。

### GP 与 LLM 的可能分工

- GP：在给定空间内启发式搜索，速度快，无先验方向
- LLM：理解经济学逻辑，缩小搜索范围，有先验但搜索效率低
- 可能的组合：LLM 缩小搜索空间 → GP 在其中高效搜索

注：GP 的搜索速度取决于回测复杂度，简单 IC 计算和完整策略回测的效率差异很大。

## 七、待验证的核心假设

1. LLM 指的方向是否比随机方向的有效因子密度更高？
2. 在 LLM 缩小的搜索空间内，GP 能否找到回测有效且样本外稳健的因子？
3. 这些因子的 alpha 衰减速度如何？
4. LLM 参与迭代（如 QuantaAlpha 的方式）vs 仅提供初始方向，哪种更有效？

### 可能的验证实验

1. 随机搜索 N 个因子 → 有效比例 p1
2. LLM 指方向后搜索 N 个因子 → 有效比例 p2
3. 比较 p1 和 p2

（N 的大小、"有效"的定义、回测方法等细节需要在实验设计时确定）

## 八、已有相关工作

| 项目 | 方法 | 关键结果 |
|------|------|----------|
| QuantaAlpha (2026) | LLM + 进化策略，轨迹级变异和交叉 | IC=0.1501 (GPT-5.2), 年化 27.75% |
| Alpha-GPT (2025) | LLM 作为研究员和因子搜索的中介 | WorldQuant 锦标赛前 10 (41000 队) |
| AlphaAgent (2025) | LLM + 正则化探索对抗 alpha 衰减 | DeepSeek-R1 表现最优 |

这些工作表明 LLM + 进化搜索的方向已有初步验证，但仍处于早期阶段。
